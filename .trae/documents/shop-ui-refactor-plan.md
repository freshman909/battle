# 兵工厂商店UI重构计划

## 目标
根据用户提供的截图，将商店风格统一为战斗场景风格，并优化对话框位置。

## 当前问题
1. 商店使用Canvas绘制，风格为木纹手绘风，与战斗场景（深色+圆角卡片）不一致
2. 魔法师对话框位于右侧，与商品货架重叠
3. 需要统一为第一张图片的深色背景+圆角卡片风格

## 设计方案

### 整体风格（参考战斗场景）
- **背景**: 深色渐变背景（#0f172a 到 #1e293b），与战斗场景一致
- **卡片**: 圆角矩形（rounded-lg），半透明深色背景（bg-slate-800/80）
- **边框**: slate-600 细边框
- **文字**: 白色主文字，黄色/蓝色强调色
- **按钮**: 圆角按钮，绿色购买/灰色返回

### 具体修改

#### 1. 背景风格修改
- 将木纹背景改为深色渐变（与Battlefield一致）
- 添加细微的网格纹理或点状纹理
- 使用 slate 色系替代 brown 色系

#### 2. 魔法师对话框位置
- 将对话框从右侧移到魔法师头顶上方
- 对话框宽度适应，不超出画布边界
- 使用绝对定位计算：wizardX - dialogWidth/2, wizardY - dialogHeight - 20

#### 3. 商品卡片风格
- 改为圆角矩形（radius: 8px）
- 背景色：slate-800/80（半透明）
- 边框：1px solid slate-600
- 选中状态：黄色边框 + 发光效果
- 悬停状态：背景变亮

#### 4. 货架风格
- 移除木质货架外观
- 改为简洁的网格布局
- 使用gap分隔商品

#### 5. 按钮风格
- 购买按钮：绿色圆角（与当前一致）
- 返回按钮：灰色圆角（与当前一致）
- 添加hover效果
